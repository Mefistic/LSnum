Documentation

The syntax is very simple, everything is based on this:
LS(OPERATION, NUMBER1, NUMBER2)

Operations

New:
Makes a new LS number object.
Usage:
LS('new', '1.8e308')
LS('new', '1.8e-308')

Add:
Adds two values.
Usage:
LS('add', 'x1', 'x2') = x1 + x2

Sub:
Subtracts the first value from the second.
Usage:
LS('sub', 'x1', 'x2') = x1 - x2

Mul:
Multiplies two values.
Usage:
LS('mul', 'x1', 'x2') = x1 * x2

Div:
Divides the first value by the second.
Usage:
LS('div', 'x1', 'x2') = x1 / x2

Pow:
Raises the first value to the power of the second value.
Usage:
LS('pow', 'x1', 'x2') = x1 ^ x2

Root:
Takes the root of the first value with a root index of the second value.
LS('root', 'x1', 'x2') = x2 âˆš x1

Log:
Takes the log of the first value with a log base of the second value.
LS('log', 'x1', 'x2') = log_x2(x1)

Floor / Round / Ceil:
Rounds down, normally or up a value.
LS('floor', '1.1') = 1
LS('round', '1.1') = 1
LS('round', '1.9') = 2
LS('ceil', '1.1') = 2

Max / Min:
Gets the maximum or minimum of two values.
LS('max', '1', '2') = 2
LS('min', '1', '2') = 1

Lt / Lte / Eq / Gt / Gte:
Returns a boolean (true/false) comparing if it's:
Less Than: LS('lt', '1', '2') = x1 < x2 = true
Less Than or Equal To: LS('lte', '1', '2') = x1 < x2 = true
Equal to: LS('eq', '1', '2') = x1 == x2 = false
Greater Than: LS('gt', '1', '2') = x1 < x2 = false
Greater Than or Equal To: LS('gte', '1', '2') = x1 < x2 = false
